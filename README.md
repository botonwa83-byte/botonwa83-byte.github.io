# King Top 教育 App 个人站

这是一个可直接发布到 GitHub Pages 的静态个人网站，用于展示教育 App 产品矩阵：

- WordPulse
- MathApex
- PhysApex
- ChemApex
- BioApex
- ChinApex
- PolApex

## 发布到 GitHub Pages

推荐建一个个人站仓库：

```text
botonwa83-byte.github.io
```

把本目录里的 `index.html` 和 `assets/` 复制到该仓库根目录，提交并推送。GitHub Pages 会把个人站发布到：

```text
https://botonwa83-byte.github.io/
```

也可以把本目录内容放进任意仓库的 `docs/` 目录，通过仓库 Pages 发布到：

```text
https://botonwa83-byte.github.io/<repo-name>/
```

## 后续建议

- App Store 上架后，在产品卡片里补真实 App Store 链接。
- 保留每个 App 独立的隐私政策、用户协议和技术支持页，不要用个人主页替代合规链接。
- PolApex 目前只有方案文档，页面中已标注为“规划中”。
